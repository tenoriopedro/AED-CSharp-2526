# Semana 01 - Máquina de Calcular (v1)

Primeira iteração do projeto da Máquina de Calcular, desenvolvida em ambiente de Consola.

## 🎯 Objetivos da Aula
* Compreensão da sintaxe básica de C#.
* Leitura de dados do utilizador via linha de comandos (`Console.ReadLine`).
* Conversão de tipos de dados (Strings para numéricos).
* Implementação de lógicas condicionais (`if`/`switch`) para operações aritméticas.

## 🚀 Como Executar
1. Abra o ficheiro `MaqCalcular_v1.sln` no Visual Studio.
2. No **Solution Explorer** (painel à direita), clique com o botão direito do rato sobre o projeto `MaqCalcular_v1` e selecione **Set as Startup Project**.
3. Prima `F5` para compilar e iniciar a aplicação de consola.
