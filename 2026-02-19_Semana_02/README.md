# Semana 02 - Máquina de Calcular (v2)

Evolução da Máquina de Calcular, marcando a transição da linha de comandos para uma Interface Gráfica de Utilizador (GUI).

## 🎯 Objetivos da Aula
* Introdução ao desenvolvimento *Desktop* com **Windows Forms**.
* Desenho da interface: manipulação da *Toolbox* (Botões, Caixas de Texto, Labels).
* Programação orientada a eventos: associar lógicas de cálculo aos eventos padrão de `Click` dos botões.

## 🚀 Como Executar e Editar
1. Abra o ficheiro `MaqCalcular_v2.sln` no Visual Studio.
2. Para visualizar a interface gráfica antes de compilar: no **Solution Explorer**, faça duplo clique sobre o ficheiro `Form1.cs` (ou selecione-o e prima `Shift+F7`) para abrir o **Form Designer**.
3. Prima `F5` para compilar o projeto e iniciar a aplicação.
